1. 경험 셀렉션 [ 나의 경험 -> 회사가 뽑아야하는 이유를 알리자 ]
* 성공하는 사람 : 필살기 3~5개를 기업에 맞게 적용한다 [기업에따라 변화 / 순서 + 강조하는 사항]
* 실패하는 사람 : 하고싶은말 1~2게 쓰고 지원하는 회사마다 복붙하여 적용
경험 : 프로젝트 역할 성과 -> 10 개이상 작성 그중에서 5개 선택
추천 : 엘리베이터 테스트 해보기[킵]

2. 마케팅 관점 : 경험이 아무리 좋아도 면접관이 관심없으면 X
- 나의 관점에서 말하려 하지말고, 면접관의 방법으로 표현하지 않으면 면접관이 선택하기 어렵다.
유사경험(시작) -> 성공경험() -> 인사이트

나의경험은 한정되어 있고 정해져 있는데 회사/면접관가 원하는 경험을 어떻게 찾을건가?
-> 지원동기를 앞단에서부터 준비 [ 현직자 인터뷰 / 경제신문 스크랩(or 산업과 기업에 대한 정보 수집 ) ] -> 모은 데이터를 바탕으로 나의 경험을 연결하는 방범, 키워드, 전달방향 찾을 수 있다.

유튜브 제목 : 면접관이 답답해서 밝히는, 합격하는 1분 자기소개! 꼭 안보고 떨어지더라...[면접왕 이형]
1분 자기소개 정리 - 이론 중심 = 50초 +- 5초
=> 면접의 전략 => 일종의 가이드라인을 친다
What
- 지원자 입장 : 1분 자기소개 하는 입장
- 면접관 입장 : 이력서 서류검토하는 입장 [ 질문할 내용 찾는 시간 ]
 + 지원자의 첫인상 알아보는 시간
Why
얼마나 중요한가?[지원자] : What에서 알아본 면접전략 + 서류검토 하고 나온 질문이 여기에 해당

How
방법[1] 구조를 지킨다
-> 핵심 내용만 두괄식으로 사용한다.[여기서만 물어봐 주세요] - 외에는 싹다 제건
-> 필살기 2개만 던진다(유사경험 + 성공경험 + 인사이트) - 이 직무를 잘할수 있는 경험 던지기[How + Result]
