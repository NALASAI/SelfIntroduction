- POST방식과, UPDATE방식에서 시간문제는 어떻게 해결했는가?
* try-catch문을 사용하여 문제가 발생한 원인 검색 -> 검색 결과 없음 
* 1번방법 : 자동으로 들어가는 내용이니 시간 입력 안하고 실행 -> null값이 들어가면 안된다는 오류 발생
* 2번방법 : Date 타입으로 시간을 받으려 했으나 timeStamp와 다른타입이라는 오류 발생
* 3번방법 : LocalDateTime으로 시간 받은 후 TimeStamp형식으로 변환 뒤 삽입하는 방법 실행 -> 컬럼명 'name'이 null값이면 안된다는 오류 발생
-> 원인 분석을 위해 service, entity, dto영역 확인 후 dto영역의 name변수명이 languageName으로 설정되어 발생한 오류임을 확인
-> name으로 수정후 재실행 -> 값이 제대로 삽입됨
